# Schedule Reminder

A simple desktop app to create custom daily reminders with notifications.
Built with Electron + JavaScript as a learning project.

## Features
- Add/edit/delete reminders with custom name, info, and time
- Get a native desktop notification when it's time
- Mark reminders as done so they stop notifying for the day

## Tech stack
- Electron
- Node.js

## Setup
\`\`\`
npm install
npm start
\`\`\`
